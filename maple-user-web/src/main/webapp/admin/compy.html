平安普惠简介

平安普惠金融业务集群（以下简称“平安普惠”）为 联营企业的下属业务集群。主要经营个人和企业信用贷款相关工作。
秉承“信任就是力量”的品牌理念，以创新的科技和卓越客户体验为广大小微型企业和个人客户提供更加优质的贷款服务，打造全球贷款客户信赖的消费金融品牌。
平安普惠拥有超过4万人的线下销售和服务团队，分布于全国200余个城市超过570家网点。线上超过5000人的远程销售服务团队，7*24小时为个人及小微企业客户提供专业的消费金融服务，并为客户提供愉悦、简单的贷款互动体验；强大的外部商务合作能力，构建起贷款生态系统。
